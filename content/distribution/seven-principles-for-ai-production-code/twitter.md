## Post 1

If you want AI to produce production-ready code, the model matters less than the workflow around it.

---

## Post 2

The seven principles I trust most:

1. Rich context beats clever wording
2. Understanding before implementation
3. Scenarios are the contract
4. Feedback loops > models
5. Write the shape before the code
6. Review behavior first
7. Concurrency is earned

---

## Post 3

Underneath all seven is the same pattern:

- make intent explicit
- make feedback fast
- make review disciplined

That operating system matters more than most prompt tricks.

---

## Post 4

A strong model with weak feedback still drifts.
A decent model with strong feedback often converges.
