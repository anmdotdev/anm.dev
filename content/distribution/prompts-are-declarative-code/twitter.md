## Post 1

A useful reframe for AI coding:

prompts are starting to function like declarative code.

---

## Post 2

The important prompt is not:
"build me a dashboard"

It is closer to a spec:

- relevant files
- intended behavior
- constraints
- scenarios
- boundaries

---

## Post 3

At that point, the prompt is no longer fluff.

It is doing engineering work.

It narrows the solution space and tells the model what kind of implementation is allowed to exist.

---

## Post 4

Code still matters.
It remains the executable layer.

But prompts, specs, scenarios, and tests increasingly shape the layer above it.
