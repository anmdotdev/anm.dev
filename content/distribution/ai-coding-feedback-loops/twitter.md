## Post 1

A lot of "AI wrote bad code" stories are really feedback-loop stories.

The model was coding blind.

---

## Post 2

If the agent cannot:

- run the app
- inspect the browser
- read the console
- trace requests
- dry-run scenarios

then it is not debugging.
It is guessing.

---

## Post 3

The useful mental model is not:
"ask for code and judge the result."

It is:
"direct an implementation loop."

Define behavior.
Implement.
Inspect.
Correct.
Review.

---

## Post 4

Bad AI output is very often just blind output.

Once you complete the loop, a decent model can converge surprisingly well.
